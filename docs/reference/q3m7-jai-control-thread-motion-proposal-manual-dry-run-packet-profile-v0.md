## 23. Risks and interpretive guardrails

- Packet profile fields must not be copied into implementation, schema, parser, API, database, migration, runtime handler, provider dispatch, or route execution without a separate CONTROL_THREAD route.
- Operator prompt packet must remain manual context and not approval.
