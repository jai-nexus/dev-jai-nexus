# Proposal: Agent Governance Sandbox Prototype v0

## Purpose

Add the first fixture-only sandbox prototype for future agent-governance
drafting, voting, and ratification review.

## Scope

- add canon defining sandbox prototype categories and constraints
- tie the prototype to the sandbox boundary and existing drafting/voting designs
- keep fixture outputs review-only and non-canon by default

## Non-goals

- no live drafting
- no live voting
- no autonomous ratification
- no provider/model dispatch
- no runtime execution
- no Corpus V2 opening or numbering reset
