# Proposal: Cross-Repo Canonical Timeline and Golden Script Routing v0

## Purpose

Create a planning/canon package in `dev-jai-nexus` that maps cross-repo canon
ownership and routing for the Q2M5-Q4M12 timeline, Golden Script, NHID,
`.nexus` governance spine, and JAI Toolchain product-foundry layer.

## Scope

- add a cross-repo roadmap timeline for Q2M5-Q4M12
- add cross-repo canon routing references
- define Golden Script as a cross-repo operating ritual
- define NHID as durable project-intelligence coordinate system
- define `.nexus` as a governance spine across repos/domains
- define JAI Toolchain as the product-foundry layer
- treat `audit-nexus` as first-class
- tie Q2M5 operator-loop work to the larger roadmap
- refresh the bundled motion snapshot through `motion-0194`

## Non-goals

- no mutation of any other repo
- no runtime behavior change
- no authority expansion
- no provider/model calls
- no execution, branch-write, PR, scheduler, automation, or persistence
- no declaration that `dev-jai-nexus` owns all cross-repo canon
