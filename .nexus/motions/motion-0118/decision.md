# Decision: Corpus V1 Program Planning Canon v0 — pre-motion planning, program graph, context inheritance, and launch packets

**Motion:** motion-0118
**Kind:** documentation

## Status

DRAFT

## Summary

Establishes the Corpus V1 Program Planning Canon v0: the governed reference
for how strategic motion lines are planned, structured, decomposed, and
launched in dev-jai-nexus. This is the final major Corpus V1 foundation move
before the Corpus V2 transition.

The canon covers: the 9-phase pre-motion planning workflow (goal framing
through launch packet), program graph vocabulary (era, epic, program, motion
line, parent/child relationships), context inheritance rules for motion
lines, motion quality standards, and the launch packet template.

Implementation produces 7 artifacts across `.nexus/docs/` and a new
`.nexus/programs/` directory. No runtime, UI, DB, or portal changes. No
prior motions modified.

After ratification, the repo will have:
- A planning canon that makes strategic arc authoring repeatable
- A quality standard that future JAI Agent panels can evaluate motions against
- A program graph capturing the Corpus V1 arc of work
- A launch packet template that opens each new planned motion line cleanly

This decision.md will be updated to RATIFIED after implementation is complete,
gate validation passes, and the vote is cast under motion-0118.
