# Proposal: Corpus V2 Gate Evidence Fixtures v0

## Purpose

Add static sample evidence records mapped to settled Corpus V2 readiness gate
ids.

## Scope

- add illustrative evidence records for drafting shape, canonical voting shape, human boundary shape, snapshot posture, and blocked authority posture
- map the examples to existing gate ids from the static readiness model
- keep all evidence examples fixture-only and non-enforcing

## Non-goals

- no machine enforcement
- no claim that fixtures satisfy gates by themselves
- no authority expansion
- no Corpus V2 opening
