# Proposal: Static Vote/Ratification Trace Fixture v0

## Purpose

Add a static fixture demonstrating future canonical vote and ratification trace
shape.

## Scope

- add a local JSON fixture for a simulated vote and ratification trace
- demonstrate canonical voter identities, reasons, evidence refs, and human
  intervention
- include a concise blocked example
- keep the fixture simulated, review-only, and non-authoritative

## Non-goals

- no live voting
- no autonomous ratification
- no provider/model dispatch
- no runtime execution
- no Corpus V2 opening or numbering reset
