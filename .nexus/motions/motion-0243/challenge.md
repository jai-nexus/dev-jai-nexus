# Challenge: Corpus Namespace Snapshot Model v0

## Challenge

Defining a future corpus-aware snapshot model could be misread as a signal that
Corpus V2 is about to open or that current snapshot behavior should be changed
immediately.

## Response

The model keeps the boundary explicit:

- Corpus V2 is not opened
- numbering is not reset
- current Corpus V1 snapshot format is preserved
- snapshot additions are conceptual only
- Corpus V1 and Corpus V2 lineage remain distinct
