# Proposal: Sandbox Validation Surface Alignment v0

## Purpose

Surface static/manual validator posture on `/operator/corpus` without adding
controls or runtime behavior.

## Scope

- extend the static sandbox model with validator summary data
- add a compact read-only validator posture block to the existing sandbox section
- show command, checked categories, source files, and explicit no-enforcement posture

## Non-goals

- no run button
- no API call
- no automatic validation
- no route changes
- no page redesign
