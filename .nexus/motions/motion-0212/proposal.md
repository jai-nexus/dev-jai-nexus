# Proposal: Static Motion Draft Fixture v0

## Purpose

Add a static fixture demonstrating the future shape of an internally drafted
Corpus V2 motion package.

## Scope

- add a local JSON fixture for a simulated draft motion
- expose the fixture through a compact static UI model if helpful
- keep the fixture example-only, not canon, and not executable

## Non-goals

- no live draft
- no branch-write or PR authority
- no provider/model dispatch
- no runtime execution
- no Corpus V2 opening or numbering reset
