# Proposal: Sandbox Failure Trace Fixtures v0

## Purpose

Add explicit simulated blocked and failure-mode traces for sandbox review.

## Scope

- add failure traces for authority, validation, human review, noncanonical voter, and provider-dispatch blockers
- label all examples as fixture-only, simulated, non-authoritative, and review-only
- surface concise failure examples in the existing sandbox review area

## Non-goals

- no real failure detection
- no runtime rejection
- no live drafting or voting
- no Corpus V2 opening
