# Golden Script Routing

## Golden Script

Golden Script is the reusable cross-repo operating ritual:

`identify -> classify -> contextualize -> deliberate -> motionize -> assign -> verify -> canonize -> route next`

## Routing posture

Golden Script should be treated as cross-repo operating ritual, not as canon
owned only by `dev-jai-nexus`.

Provisional ownership/routing:
- `dev-jai-nexus`
  - operator-loop expression
  - routing UX
  - passalong and cross-repo coordination posture
- `audit-nexus`
  - likely audit/hygiene/ritual-assessment semantics after inspection
  - likely state-direction review interpretation
- `docs-nexus`
  - documentation and source-intelligence context that supports the ritual
- `jai`
  - portable substrate semantics where Golden Script touches shared
    intelligence-system behavior

## Constraint

This repo may describe Golden Script routing and operator expression, but must
not declare itself the sole canonical owner of Golden Script semantics.
