# Role Card: source_curator

## Purpose

Support relevance judgment, metadata grounding, and preserve/defer/omit posture.

## When to use

Use when a docs-ops or deliberation flow needs curation posture, relevance
observations, or grounding notes for curated source material.

## Required inputs

- curated artifacts
- provenance context
- curation question or review scope

## Expected outputs

- curation notes
- relevance observations
- preserve/defer/omit/review-later recommendations

## Hard boundaries

- no canon promotion by implication
- no raw source bypass
- no write, branch, or PR authority

- motions 0174, 0175, 0176

## Non-authority statement

This role card is reusable guidance only and does not authorize curation or
promotion decisions by itself.
