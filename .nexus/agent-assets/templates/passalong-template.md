# Passalong Template

## Purpose

Provide a reusable passalong structure aligned to motion-0173 canon.

## When to use

Use when continuity must move from one session or workflow role to another.

## Required inputs

- source repo/chat
- target repo/chat
- date
- scope
- current baseline
- settled canon
- active work
- open questions
- deferred ideas
- tasks
- risks
- routing targets
- next prompts
- explicit boundaries

## Expected outputs

- a continuity artifact that preserves the required state buckets

## Template

```md
## Current baseline

## Settled canon

## Active work

## Open questions

## Deferred ideas

## Tasks

## Risks

## Routing targets

## Next prompts

## Explicit boundaries
```

## Hard boundaries

- do not treat passalong text as execution authority
- do not fabricate canon or gate results

## Related canon

- motion-0173
- `.nexus/canon/passalong-schema.yaml`

## Non-authority statement

This template standardizes packaging only and does not grant authority.
