# PR Description Template

## Purpose

Provide a reusable summary structure for review-facing PR descriptions.

## When to use

Use when a seam needs a concise review package or PR draft description.

## Required inputs

- summary
- rationale
- changed or proposed files
- validation
- guardrails
- non-goals

## Expected outputs

- a clear PR description draft

## Template

```md
## Summary

## Rationale

## Files

## Validation

## Guardrails

## Non-goals
```

## Hard boundaries

- do not imply that the PR exists if it does not
- do not omit material guardrails

## Related canon

- motion-0173
- motion-0176

## Non-authority statement

This template is review packaging only and does not create a PR.
