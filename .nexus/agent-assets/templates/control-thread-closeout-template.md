# Control Thread Closeout Template

## Purpose

Provide a reusable closeout structure for handing completed seams back to
CONTROL_THREAD.

## When to use

Use when a bounded seam is complete enough to summarize continuity, risks, and
next routing options back into control-thread context.

## Required inputs

- current baseline
- settled canon
- completed or active work
- open questions
- deferred items
- validation status
- next prompts

## Expected outputs

- a compact control-thread closeout artifact

## Template

```md
## Current baseline

## Settled canon

## Active or completed work

## Open questions

## Deferred items

## Validation status

## Next prompts
```

## Hard boundaries

- do not claim sync-back without supporting evidence
- do not invent closure status
